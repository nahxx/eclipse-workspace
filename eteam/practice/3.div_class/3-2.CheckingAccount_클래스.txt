패키지에 CheckingAccount 클래스 생성하기

1. 현재 객체에서만 접근가능하도록 아래 변수 생성
	1-1. 계좌번호, 잔액, 고객정보, 대출 한도를 담을 변수 생성하기
	      단, 잔액과 대출 한도는 실수값이 들어간다.

2. 계좌번호, 잔액, 대출 한도를 파라미터로 가지는 생성자를 생성하고 초기화하기

3. 메서드 생성하기
	3-1. 입금할 금액을 받아 구현하는 입금 메서드 생성
	      조건)
	      마이너스 대출 중일 경우 잔액과 입금금액이 한도를 넘어서면 대출상황을 원상복귀 하고 남은 잔액은 더해준다.

	3-2. 출금할 금액을 받아 구현하는 출금 메서드 생성
	      조건)
	      출금하려고 할 때 잔액이 모자라다면 대출 한도 금액을 가져와서 출금처리
	      한도를 더해도 금액이 모자라다면 "잔고부족" 출력

	3-3. 고객정보, 계좌번호, 잔액, 대출 한도 get & set 메서드 생성
