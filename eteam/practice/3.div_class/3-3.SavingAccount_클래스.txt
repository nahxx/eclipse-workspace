패키지에 SavingAccount 클래스 생성하기

1. 현재 객체에서만 접근가능하도록 아래 변수 생성
	1-1. 계좌번호, 잔액, 고객정보, 이자율 담을 변수 생성하기
	      단, 잔액과 이자율은 실수값이 들어간다.

2. 계좌번호, 잔액, 이자율을 파라미터로 가지는 생성자를 생성하고 초기화하기

3. 메서드 생성하기
	3-1. 입금할 금액을 받아 구현하는 입금 메서드 생성

	3-2. 출금할 금액을 받아 구현하는 출금 메서드 생성
	      만약 잔고가 부족하다면 "잔고부족" 출력

	3-3. 고객정보, 계좌번호, 잔액, 이자율 get & set 메서드 생성
