$02_add_variable 패키지 안의 Account 클래스에 추가하기

1. 변수 생성
	1-1. 현재 객체에서만 접근가능하며 '이자율'을 실수형 type으로 가지는 interestRate 변수
	1-2. 현재 객체에서만 접근가능하며 '마이너스 대출 한도'를 실수형 type으로 가지는 overdraftAmount 변수

2. 생성자 생성(오버로딩)
	2-1. 계좌번호, 잔액, 이자율을 매개변수로 가지며, 매개변수로 받은 값으로 멤버변수를 초기화하는 생성자 생성
	2-2. 계좌번호, 잔액, 마이너스 대출 한도를 매개변수로 가지며, 매개변수로 받은 값으로 멤버변수를 초기화하는 생성자 생성
	2-3. 2-2에서 에러가 나는 이유는?
		=> 매개변수의 타입 순서와 개수가 똑같고 매개변수의 이름만 바뀔 경우 -> 생성자 오버로딩X
		=> 매개변수 타입의 순서가 다르지만 개수가 똑같은 경우 => 생성자 오버로딩X
	2-4. 해결방법 찾아서 생성자 오버로딩 해보기
		Hint ) 2-1에서 생성한 생성자와 2-2에서 생성한 생성자를 합쳐보기

3. 메서드 생성
	3-1. get-set 메서드
		3-1-1. 이자율 getter & setter
		3-1-2. 마이너스 대출 한도 getter & setter


※생성자 오버로딩
=>1개의 클래스에서 동일한 이름의 생성자를 여러개 정의하는 것

오버로딩 조건
=>매개변수의 자료형, 개수, 순서를 다르게 해야한다.

오버로딩 이유
=> 동일한 기능의 생성자를 정의할 때 다양한 매개변수값을 받아 초기화하기 위해서다.
